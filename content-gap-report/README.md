# گزارش نواقص محتوایی سایت

## توضیح

این پوشه حاوی گزارش کامل مقایسه محتوای **جزوه مرجع** (`unified_book.md`) با **محتوای فعلی سایت** است.

هدف: شناسایی تمام مطالبی که در جزوه وجود دارد اما در سایت پوشش داده نشده یا ناقص ارائه شده است.

## فایل‌های گزارش

| فایل                  | محتوا                                                    |
| --------------------- | -------------------------------------------------------- |
| `01_gap_step0.md`     | نواقص گام ۰ (موتور ریاضی)                                |
| `02_gap_step1.md`     | نواقص گام ۱ (رفتار مصرف‌کننده)                           |
| `03_gap_step2.md`     | نواقص گام ۲ (رفتار تولیدکننده)                           |
| `04_gap_step3.md`     | نواقص گام ۳ (بازارها)                                    |
| `05_gap_step4.md`     | نواقص گام ۴ (الیگوپولی)                                  |
| `06_verified_gaps.md` | مواردی که توی گزارش نیومده و جا مونده بودن. اپدیت گزارش. |

## روش بررسی

جزوه خط به خط از صفحه ۱ تا ۱۲۶ بررسی و با محتوای سایت مقایسه شد.
