# Project: جزوه شب امتحانی جامع اقتصاد خرد پیشرفته

## Architecture
پروژه یک وب‌سایت کاملاً استاتیک React است که با استفاده از Vite بیلد می‌شود.
ساختار مسیریابی (Routing) در `src/App.jsx` با استفاده از `react-router-dom` پیاده‌سازی شده است.
هر گام به عنوان یک کامپوننت React در مسیر `src/pages/StepX.jsx` ایجاد شده و در `src/App.jsx` ایمپورت شده و به مسیر مربوطه متصل می‌گردد.
نمایش فرمول‌های ریاضی با استفاده از پکیج‌های `katex` و `react-katex` انجام می‌شود.

## Code Layout
- `src/App.jsx` — هاب اصلی مسیریابی و قالب‌بندی سایت
- `src/index.css` — استایل‌های پایه و متغیرهای تم تاریک/روشن پروژه
- `src/pages/Step0.jsx` — موتور ریاضیات اقتصاد
- `src/pages/Step1.jsx` — رفتار مصرف‌کننده
- `src/pages/Step2.jsx` — رفتار تولیدکننده (قانون آینه)
- `src/pages/Step3.jsx` — ساختارهای بازار کلاسیک و تحلیل مالیات‌ها
- `src/pages/Step4.jsx` — انحصار چندجانبه و استراتژیک به همراه جدول مقایسه‌ای ماتریسی
- `public/svg/` — محل نگهداری تمام فایل‌های SVG بهینه‌شده برای تم تاریک

## Interface Contracts
- هر کامپوننت صفحه (`StepX.jsx`) باید یک کامپوننت React به عنوان default export صادر کند.
- مسیرهای صفحات در `src/App.jsx` عبارتند از: `/step0` و `/step1` و `/step2` و `/step3` و `/step4`.
- تمامی فایل‌های SVG کپی‌شده از مسیر `jozve-marja/final-md` به `public/svg` باید در کدهای صفحات با آدرس استاتیک `/svg/<filename>.svg` ارجاع داده شوند و رنگ‌های آن‌ها برای تم تاریک (تغییر متون و خطوط تیره به روشن/سفید یا رنگ مناسب) بهینه‌سازی شود.
